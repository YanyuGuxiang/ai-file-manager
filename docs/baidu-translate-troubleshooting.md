# 百度翻译API配置说明

## 错误代码含义

如果在日志中看到类似以下错误：
```
翻译API返回错误 [52003]: UNAUTHORIZED USER
```

这通常表示以下几种情况之一：
1. APP ID 或 密钥配置错误
2. 账户未开通翻译服务
3. 账户余额不足
4. IP地址不在允许列表中（如设置了IP白名单）

## 正确配置步骤

### 1. 申请百度翻译API
1. 访问 [百度翻译开放平台](https://api.fanyi.baidu.com/)
2. 注册账号并登录
3. 进入控制台，创建应用
4. 获取APP ID和密钥

### 2. 配置环境变量
在项目根目录的 `.env` 文件中添加：

```env
BAIDU_TRANSLATE_APPID=your_actual_app_id_here
BAIDU_TRANSLATE_SECRET=your_actual_secret_key_here
```

### 3. 检查服务状态
1. 登录百度翻译开放平台管理后台
2. 确认服务是否已开通
3. 检查账户余额是否充足
4. 如需设置IP白名单，请将当前服务器IP添加进去

### 4. 常见错误代码

- `52003`: 未授权的用户（请检查APP ID和密钥是否正确）
- `54001`: 签名错误（请检查密钥是否正确）
- `52001`: 请求超时
- `52002`: 系统错误
- `58001`: 译文语言不支持

## 验证配置

重启应用后，检查日志中是否仍有错误信息。如果配置正确，应该能看到：
```
翻译函数: API调用成功，译文: ...
```

## 故障排除

如果仍有问题：
1. 删除 `translation-cache.json` 文件以清除旧的缓存
2. 检查网络连接是否正常
3. 确认当前时间与服务器时间同步（时间戳用于签名生成）
4. 参考官方文档：https://api.fanyi.baidu.com/doc/21
